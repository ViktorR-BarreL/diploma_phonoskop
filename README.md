# [ф]оноскоп 2026

Интеллектуальный инструмент для автоматического фонетического анализа речи, создания разметки и поиска звуковых сочетаний.

## Описание
**[ф]оноскоп** — десктопное приложение для автоматизации работы лингвистов и экспертов-фоноскопистов. Система преобразует аудио в последовательность фонем с точной привязкой ко времени, предоставляет инструменты для ручной коррекции, поиска паттернов и экспорта данных.

## Технологический стек
- **Интерфейс**: PyQt6, PyQtGraph (осциллограммы, спектрограммы, LOD-оптимизация)
- **Обработка аудио**: Librosa, SoundFile, Wave
- **Нейросети**: PyTorch (CUDA), Transformers (Wav2Vec2ForCTC), Vosk API
- **Данные**: SQLite3, JSON
- **Экспорт**: TextGrid / PitchTier (Praat), PDF (QTextDocument), CSV

## Первичная настройка

Выполните эти команды последовательно для полной настройки проекта:
### 1. Клонирование и окружение
```bash
# Клонирование репозитория
git clone https://github.com/ViktorR-BarreL/diploma_phonoskop.git
cd diploma_phonoskop

# Создание и активация виртуального окружения
python -m venv venv
# Для Windows:
venv\Scripts\activate
# Для Linux/macOS:
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt
```

### 2. Подготовка структуры для моделей
Программа ожидает наличие весов нейросетей в папке `models`. Создадим нужные директории:
```bash
mkdir models
mkdir models/phonoscopic
mkdir models/vosk
```

## Загрузка моделей
Для работы приложения необходимо скачать и распаковать модели:

1.  **Фонемная модель (Wav2Vec2)**: 
    *   Скачайте веса с [Hugging Face](https://huggingface.co/ViktorR-BarreL/phonoscopic)
    *   Распакуйте содержимое в папку `./models/phonoscopic/`
2.  **Модель Vosk (для текста)**:
    *   Скачайте русскую модель с [официального сайта Vosk](https://alphacephei.com/vosk/models) (рекомендуется `vosk-model-small-ru-0.22`)
    *   Распакуйте содержимое в папку `./models/vosk/`
  
## Запуск приложения
После настройки моделей запустите главный файл:
```bash
python main.py
```

## Основные возможности
- **Автоматическая разметка фонем**: Модель `Wav2Vec2` выравнивает звуки с точностью до миллисекунд.
- **Интегрированный STT**: Текстовая расшифровка с помощью `Vosk` (редактируемая).
- **Умный поиск**: Поиск по паттернам (СГС, ударные гласные) или пользовательским цепочкам `[ф][о][н]`.
- **LOD Визуализация**: Быстрая отрисовка графиков даже на очень длинных записях.
- **Профессиональный экспорт**: 
    - **Praat (TextGrid)** с поддержкой разделения по слоям (Tiers).
    - **PitchTier** для интонационного анализа.
    - **PDF-отчеты** и табличные данные (CSV/TXT).

## Демонстрационный проект
В репозиторий включена папка `demo_project`. При первом запуске программа автоматически зарегистрирует проект **"ДЕМО: Пример разметки"**, чтобы вы могли сразу протестировать функции визуализации и поиска.
